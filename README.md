## 📋 What is warpweb?

warpweb is a simple application that lets you convert, change, and analyze files through a web interface. It builds on FileWarp to give you access to powerful file tools without needing complicated software. You can work with images, videos, audio, and other file types right from your web browser.

The app runs on Windows and uses common technologies like Django for the backend and React with Tailwind CSS for the user interface. These technologies make it fast and easy to use, even if you don't have programming skills.

warpweb lets you:

- Change file types (image, video, audio)
- Extract audio from videos
- Animate images using simple effects
- Analyze file details like size and format

This guide will help you download and run warpweb on your Windows computer step-by-step.
